787. K 站中转内最便宜的航班

  • 题目

    有 n 个城市通过一些航班连接。给你一个数组 flights ，其中 flights[i] = [fromi, toi, pricei] ，表示该航班都从城市 fromi 开始，以价格 pricei 抵达 toi。

    现在给定所有的城市和航班，以及出发城市 src 和目的地 dst，你的任务是找到出一条最多经过 k 站中转的路线，使得从 src 到 dst 的 价格最便宜 ，并返回该价格。 如果不存在这样的路线，则输出 -1。

示例 1：

输入:
n = 3, edges = [[0,1,100],[1,2,100],[0,2,500]]
src = 0, dst = 2, k = 1
输出: 200

示例 2：

输入:
n = 3, edges = [[0,1,100],[1,2,100],[0,2,500]]
src = 0, dst = 2, k = 0
输出: 500

  • 难度：中等

  • 分类：多看

  • 题解

    解这道题，最好的理解方式还是通过例子来介绍。首先我们要知道，k为中转城市个数，那么我们中专k个城市，需要飞行k+1次。比如中转1个城市，那么出发城市->中转城市->目的地城市，相当于我们飞了2次。搞懂了这个之后，我们用示例1来讲解这个题：

    一共有三个城市，那么我们可以声明下面这个数组储存数据，数组为n维的。

    这个数组是用来储存出发地src到各个城市的距离的，所以首先初始化各个距离为max，max为Integer类型所能储存最大的数。

    次数 0 1 2 3
    0 max max max max

    然后src->src的距离为0，所以我们将这个距离设为0：

    次数 0 1 2 3
    0 0 max max max

    然后我们遍历一次数组，找到出发地是src的数据，更新数组：

    次数 0 1 2 3
    0 0 100 500 max

    这样我们就完成了整个数组的初始化，我们开始第二次遍历，首先也是初始化数组

    次数 城市0 城市1 城市2 城市3
    0 0 100 500 max
    1 0 max max max

    这时，我们遍历数组，首先我们看到第一条数据[0,1,100]，是城市0到城市1的距离100，这个时候我们比较上一次（0次时）城市src到城市0的距离（0->0距离为0）+100与次数1直接到城市1的距离，我们能够得到（0+100<max），所以我们的赋值如下：

    次数 城市0 城市1 城市2 城市3
    0 0 100 500 max
    1 0 100 max max

    然后我们看第二条数据：[1,2,100]，表示从城市1到城市2距离100，这时我们比较上一次（0次时）城市src到城市1的距离(0->1距离为100)+100与次数1直接到城市2的距离(100+100<max)，所以我们赋值如下：

    次数 城市0 城市1 城市2 城市3
    0 0 100 500 max
    1 0 100 200 max

    然后我们看到第三条数据：[0，2，500]，表示从城市0到城市2的距离500，这时我们比较上一次（0次时）城市src到城市0的距离（0->0距离为0）+500与次数1直接到城市2的距离，我们能够得到（0+500>200），所以我们赋值如下：

    次数 城市0 城市1 城市2 城市3
    0 0 100 500 max
    1 0 100 200 max

    那么次数1的数据就意味着经过1个中转城市，我们从src出发到各个城市的距离。我们可以知道这个数组大小为(k+1)*(n)维的，所以解这道题可以直接声明这个数组，然后按照上面的逻辑进行求解。
